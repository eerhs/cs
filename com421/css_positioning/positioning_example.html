Positioning objects with CSS

Here you'll find some examples of positioning that will help you to get things where you want them to go on your page without using tables or non-breaking spaces. Once you get going with CSS positioning, you won't need line-break tags either.

There are also rapid prototyping options like 960gs or Bootstrap that you can explore, but first let's move some things around.
